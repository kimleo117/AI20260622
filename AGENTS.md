# AI智能工具站規格檔

## 專案名稱

AI智能工具站

## 專案目標

建立一個以 Bootstrap 5 為基礎的 AI 工具介紹網站，包含首頁、關於我們、免費與付費比較、AI 工具列表與聯絡資訊，讓使用者能快速了解與比較常見 AI 工具。

## 技術規格

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Lucide Icons
- Google Analytics（網站追蹤）

## 本地端資源

- Bootstrap CSS：css/bootstrap.min.css
- Bootstrap JS：js/bootstrap.bundle.min.js
- 圖片資料夾：pic/

## 主要頁面

1. 首頁 index.html
   - 導覽列與手機版選單
   - 輪播主視覺
   - AI 工具列表與分類側邊欄
   - 聯絡表單
   - 頁尾資訊

2. 關於我們 about.html
   - 品牌故事與核心價值
   - 介面圖片與數據統計
   - CTA 按鈕連結回首頁與付費頁

3. 免費與付費 pricing.html
   - 免費方案卡片
   - 付費方案卡片
   - 方案比較與行動呼籲

## 互動功能

- Bootstrap Carousel 輪播
- 手機版導覽列收合
- 平滑捲動錨點導覽
- AI 工具卡片 hover 浮起效果
- 聯絡表單送出成功提示
- 導航與主題色彩統一

## 響應式設計

- 桌機版：使用 Bootstrap container 與 grid 排版
- 平板版：卡片以兩欄呈現
- 手機版：卡片改為單欄堆疊，輪播高度縮小

## 開啟方式

直接使用瀏覽器開啟根目錄下的 index.html，或透過 GitHub Pages 發布後瀏覽。
